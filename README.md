![a](https://i.imgur.com/SZnv42b.png)

## Run

simple example.

```sh
$ stack build
$ stack exec example
```

full components showcase.

```sh
$ stack build
$ stack exec showcase
```

Access to [localhost:3100](http://localhost:3100)
